Лабораторна робота №1: Нейронна мережа XOR

Завдання №1

Реалізація обчислювального алгоритму (Персептрон)
Логіка: XOR реалізується як композиція:
Result = OR(x1, x2) AND [NOT AND(x1, x2)]
Фінальний результат (XOR):
-

Таблиця істинності (генерована):

x1 x2 OR (y1) AND (y2) XOR (Res)

Завдання №2

Двошаровий персептрон та візуалізація
Рівняння прямих (межі нейронів):
■ x1 + x2 - 0.5 = 0 (OR поріг)
■ x1 + x2 - 1.5 = 0 (AND поріг)
Пояснення графіку:
Червоні точки: Клас 0 (False) - [0,0] та [1,1]
Сині точки: Клас 1 (True) - [0,1] та [1,0]
